# All-in-One Domain & Website Tools

An AI Skill that provides domain tools — appraisal, availability search, brand naming, and logo generation.

## Tools

| Tool | What it does |
|------|-------------|
| **Domain Appraisal** | AI valuation across 22 scoring dimensions (0-100 score + price estimate) |
| **Domain Search** | Check domain availability across 20+ TLDs |
| **Domain Discovery** | Find domains containing a keyword |
| **AI Brand Naming** | Generate brand names with available domain suggestions |
| **Logo Generation** | Create SVG logos for your brand/domain |

## Usage Examples

```
> Appraise example.com
> Is cloudai.com available?
> Find domains containing "tech"
> Help me name my AI education startup
> Generate a logo for myapp.com
```
